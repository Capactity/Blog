# 509.斐波那契数

> leetcode地址： https://leetcode.cn/problems/fibonacci-number/

**解题思路：**

例如数组：1 1 2 3 5 8

确定dp[i]含义，dp[i]：第i个斐波那契数的值

递推公式：dp[i] = dp[i-1] + dp[i-2]

dp数组如何初始化 dp[0] = 1，dp[1] = 1

遍历顺序：从前向后

即可得：

```
传入n;
if (n < 2) return n;
dp[0]=1,dp[1]=1;
for(i=2,i<=n;i++) {
  dp[i]=dp[i-1]+dp[i-2]
}
return dp[n]

// 得到遍历公式
dp[0]=d[1]
dp[1]=sum
sum=dp[0]+dp[1]
```

**复杂度：**

- 时间复杂度：*O*(*n*)。
- 空间复杂度：*O*(1)。

```
function fib(n: number): number {
    if (n<2){
        return n;
    }
    let f0=0,f1=0,f2=1;
    for (let i=2; i<=n; i++) {
        f0=f1;
        f1=f2;
        f2=f0+f1;
    }
    return f2;
};
```
