# 70.爬楼梯

> leetcode地址： https://leetcode.cn/problems/climbing-stairs/

**解题思路：**

n阶楼梯

0阶 - 1种

1阶 - 2种

3阶 - 3种

4阶 - 5种

确定dp[i]含义，dp[i]：到第i阶有多少种走法的值

递推公式：dp[i] = dp[i-1] + dp[i-2]

dp数组如何初始化 dp[0] = 1，dp[1] = 1 dp[2]=2

遍历顺序：从前向后

**复杂度：**

- 时间复杂度：*O*(*n*)。
- 空间复杂度：*O*(1)。

```
function climbStairs(n: number): number {
   if (n<=2){
        return n;
    }
    let f0=1,f1=1,f2=2;
    for (let i=3; i<=n; i++) {
        f0=f1;
        f1=f2;
        f2=f0+f1;
    }
    return f2;
};
```
